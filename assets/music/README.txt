Put your music here!

1. Copy your .mp3 files into this folder (assets/music/).
2. Open js/apps/music.js and edit the `playlist` array so each entry's
   `src` matches your filename, e.g.:

     { title: "soft focus - lofi", src: "assets/music/pocketbeats-soft-focus-aesthetic-lofi-music-background-492869.mp3" }
     { title: "soft focus - lofi", src: "tunetank-aesthetic-lo-fi-beat-348346.mp3" }

The player loops through the list. Tip: shorter/royalty-free tracks are
best for a public site.
